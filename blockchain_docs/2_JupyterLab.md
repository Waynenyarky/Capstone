# Opening, Installing, and Testing BlockchainAudit.ipynb in JupyterLab Instructions

1. **Open Anaconda Navigator**
   - Start Menu → type Anaconda Navigator → Enter

2. **Launch JupyterLab**
   - In Anaconda Navigator → click Launch under JupyterLab

3. **Go to your project folder**
   - Use the JupyterLab file browser to navigate to:
     ```
     C:\Users\erick\Desktop\BlockchainAudit
     ```
   - Make sure `BlockchainAudit.ipynb` is there

4. **Open the notebook**
   - Click `BlockchainAudit.ipynb` → it opens in a new tab

5. **Test the notebook**
   - Click **Kernel → Restart Kernel and Run All Cells**
   - Ensure all cells run without errors
